Projects

  • SatEye: Leveraging Inverse Reinforcement Learning for Autonomous Object Detection in Satellite Imagery (Publication in progress of exploring): Built an AI agent based on IRL to detect object in sattelite images.
  • Autonomous Multi-Tool LLM AI Agents for Dynamic Task Execution and User Interaction: Built an AI agent who create static websites, news checking, web scarping and sending emails leverging the R1 reasoning capabilities in DeepSeek-R1 LLM .
  • Next-Gen Text Summarization:
    • PEFT-Based Generative Dialogue Summarization: Fine-tuned FLAN-T5 with PEFT (LoRA) to generate dialogue summaries while reducing computational costs. Preprocessed datasets, optimized model weights, and evaluated performance using ROUGE and human feedback.
    • RLHF-Based Toxicity-Controlled Text Summarization: Utilized FLAN-T5 and RLHF with PPO for generating informative and non-toxic summaries. Integrated Perspective API for toxicity assessment and optimized using ROUGE metrics and reward models.
  • Machine Translation System with Locality Sensitive Hashing: Built an English-to-French translation system using word embeddings, gradient descent, and transformation matrices. Integrated LSH for efficient document similarity search and optimized retrieval.
  • Dual-Architecture Image Similarity Retrieval System: Combined ResNet50 for feature extraction with an attention mechanism to refine image representations. Leveraged FAISS for fast image indexing and implemented using PyTorch and TensorFlow.
  • Smart Home System with Speech Recognition: Developed a prototype using Arduino Uno, sensors, and Java for voice-controlled monitoring and automation of home systems. Integrated real-time status updates for temperature, motion, and door states.
  • Real-Time Sentiment Analysis of Tweets: Designed an ETL pipeline using PySpark, BERT, and logistic regression for sentiment analysis. Visualized results with PowerBI and automated data processing in Docker containers.
  • NLQ-SQL Converter: Created an interface to convert natural language queries to SQL for Oracle DB using OLLAMA and Gemma LLM. Integrated real-time schema extraction for context-aware query generation.
  • Spelling Correction System: Developed a robust spelling correction system using edit distance algorithms, probabilistic ranking, and dynamic programming. Analyzed Shakespeare's works for vocabulary and frequency-based suggestions.
  • Image Generation with GAN: Implemented adversarial training for realistic image generation on MNIST and ImageNet-MINI datasets using TensorFlow and PyTorch.
  • End-to-End 3D Scene Analysis Pipeline: Built a 3D reconstruction pipeline using SfM, MVS, and Open3D. Extracted features with SIFT and refined dense point clouds to generate accurate 3D meshes.
  • 3D Reconstruction and Classification: Designed PointNet for feature extraction and classification of point clouds, trained on datasets like ModelNet10 with PyTorch.
  • Image Segmentation with Attention and Clustering: Developed a multi-head attention encoder with K-means clustering for precise image segmentation. Reconstructed segmented images and visualized clustering results in 3D.
  • Virtual Try-On System for Footwear Personalization 3D: Implemented a 2-stage object tracking system using markerless and ICP-based tracking with Microsoft Kinect to create a virtual try-on experience.
  • Image Colorization: Designed a U-Net for converting grayscale images to color using TensorFlow and PyTorch. Trained with Adam optimizer and MSE loss for optimal results.
  • Image Processing Application: Built an application with OpenCV and Tkinter for enhancement, edge detection, and morphological processing. Integrated Hough Transform and point-of-interest detection techniques.
  • Denoising Diffusion Probabilistic Models (DDPM): Created an image generation pipeline for CIFAR10 and Sprite datasets using noise schedulers, U-Net, and sinusoidal time encoding.
  • Time Series Forecasting with Deep Learning: Designed a platform for analyzing LSTM, GRU, Transformer, and Attention-based models using PyTorch. Implemented a scalable preprocessing pipeline and real-time visualization tools.
  • Stroke Prediction with TabNet: Applied data preprocessing on Kaggle's stroke dataset and trained a TabNet model with attentive feature transformers for improved prediction accuracy.
