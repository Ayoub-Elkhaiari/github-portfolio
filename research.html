Research

  • Natural Language Query to SQL: Developed a pipeline for NLQ translation to Oracle SQL using LLMs, focusing on schema awareness and contextual accuracy.

  • Sentiment Analysis of Real-Time Data: Built an ETL pipeline for analyzing tweets using advanced NLP models like BERT.
  • Generative AI for Insurance: Fine-tuned LLMs for domain-specific tasks, such as chatbot applications in the insurance sector.
