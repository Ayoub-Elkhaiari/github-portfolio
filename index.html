About Me

Passionate AI researcher specializing in Natural Language Processing, Generative AI, and software engineering. I have experience in building intelligent systems like Retrieval-Augmented Generation (RAG) applications, text-to-SQL converters, and advanced deep learning models such as GANs and UNET.

Projects

  • RAG System for Assurance: Developed a Retrieval-Augmented Generation (RAG) system for automobile insurance, integrating FAISS for vector storage and Ollama embeddings for enhanced user query understanding.
  • NLQ to SQL Converter: Built a natural language interface to Oracle SQL using OLLAMA LLM, enabling non-technical users to query databases effectively.
  • Auto-Correct System: Designed an NLP-based auto-correct application leveraging edit distance and locality-sensitive hashing (LSH).
  • Language Translation: Implemented a semantic-similarity-based translation system using LSH for cross-language mappings.
  • UNET for Image Colorization: Designed a UNET-based model to convert grayscale images to colored ones using deep learning techniques.
  • Generative Adversarial Networks (GANs): Trained GANs for the ImageNet Mini dataset, achieving realistic image generation.
  • OCR for Insurance Documents: Built an OCR pipeline to extract structured data from multi-column PDFs using PySpark for insurance applications.
  • ETL Pipeline for Sentiment Analysis: Developed a real-time ETL pipeline to analyze sentiment in tweets using BERT, TextBlob, and logistic regression.
  • Custom LLM Embedding Framework: Created a document embedding system for semantic search using FAISS and LangChain integrations.
  • AI-Powered Debugging: Implemented a conversational AI debugging assistant to assist developers in locating and resolving code bugs efficiently.

Research Interests

My research focuses on AI-driven solutions for real-world problems, including: - Natural Language Processing for software engineering - Generative AI and Large Language Models (LLMs) - Software debugging and security through AI
